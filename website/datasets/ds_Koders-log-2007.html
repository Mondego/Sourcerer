Download page for Sourcerer Source Code Data Set: Koders-log-2007

"Koders-log-2007" is a compressed file (in 7z format) containing a Microsoft SQL Server database backup storing a yearlong usage data of Koders.com, an Internet-scale code search engine. Thanks to Black Duck Software, Inc., we are able to release this file to the research community to enable studies that will advance the understanding and state-of-the-art in large scale code search.
